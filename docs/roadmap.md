## 🎯 近期计划

### Phase 3 - 高级Agent和自动化 (规划中)

**优先级**: 高
**预计时间**: 2-3周
**目标**: 实现更智能的订阅管理和自动化能力

#### 3.1 NegotiationAgent (谈判Agent)
**业务价值**: 帮助用户争取更优惠价格

**功能**:
- [ ] 价格谈判策略生成
  - 分析用户历史订阅时长
  - 识别竞品价格
  - 生成谈判话术模板
- [ ] 客服联系信息管理
  - 保存各服务客服渠道
  - 自动生成联系邮件草稿
- [ ] 谈判结果追踪
  - 记录谈判历史
  - 成功率统计
  - 节省金额汇总

**技术方案**:
```python
class NegotiationAgent(BaseAgent):
    async def generate_negotiation_strategy(self, subscription_id: str) -> Dict:
        """生成谈判策略"""
        # 分析订阅历史
        # 查询竞品价格
        # AI生成话术
        pass

    async def draft_contact_message(self, subscription_id: str, strategy: str) -> str:
        """起草联系消息"""
        pass
```

**UI集成**:
- 在订阅详情页添加"价格谈判"按钮
- 显示谈判建议和话术
- 提供一键复制联系信息

#### 3.2 RecommendationAgent (推荐Agent)
**业务价值**: 主动发现更适合用户的服务

**功能**:
- [ ] 基于使用模式的服务推荐
  - 分析订阅使用频率
  - 识别需求缺口
  - 推荐新服务
- [ ] 替代方案推荐
  - 功能相似但更便宜的服务
  - 免费开源替代品
- [ ] 套餐升级/降级建议
  - 根据使用量建议套餐调整
  - ROI分析

**技术方案**:
```python
class RecommendationAgent(BaseAgent):
    async def recommend_alternatives(self, subscription_id: str) -> List[Dict]:
        """推荐替代服务"""
        pass

    async def analyze_plan_fit(self, subscription_id: str) -> Dict:
        """分析套餐适配度"""
        pass
```

#### 3.3 智能规则引擎增强
**当前状态**: 基础规则引擎已存在 (`core/agents/rules_engine.py`)

**增强内容**:
- [ ] 更多预设规则
  - 长期未使用自动标记
  - 价格上涨自动通知
  - 重复功能服务警告
  - 年付到期前提醒转年付
- [ ] 自定义规则构建器
  - UI界面创建规则
  - 条件组合编辑器
  - 动作选择器
- [ ] 规则执行历史
  - 记录规则触发
  - 执行结果追踪
  - 效果分析

#### 3.4 自动化任务增强
**基于**: 现有的`DailyCheckupScheduler`

**新增任务**:
- [ ] 周度成本报告
  - 每周一发送支出总结
  - 对比上周变化
  - 省钱机会汇总
- [ ] 月度优化建议
  - 每月1号深度分析
  - 年度趋势预测
  - 年度节省总结
- [ ] 价格监控
  - 定期检查服务价格变化
  - 自动标记涨价订阅
- [ ] 续费智能提醒
  - 根据重要性调整提醒频率
  - 未使用服务特殊提醒
